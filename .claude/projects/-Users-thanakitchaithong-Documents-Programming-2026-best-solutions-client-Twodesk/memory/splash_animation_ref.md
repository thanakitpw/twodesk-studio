## Splash Animation — Storyboard

7 ขั้นตอน:
1. แสดงโลโก้เต็ม "TWO DESK" — TWO เลื่อนจากซ้าย, DESK เลื่อนจากขวา
2. ตัวอักษรค่อยๆ หาย เหลือแค่ T กับ D
3. T+D transform รวมเป็นโลโก้ symbol (3D TD)
4. เพิ่มเส้นวาดรูปคนนั่ง 2 ฝั่งโต๊ะ (ลายเส้นสเก็ตช์)
5. สลับ pop up กล่องข้อความ + ก้อนเมฆบนหัวสลับ 2-3 ที เป็นรูปบ้าน ตึก คลินิก — สื่อว่ากำลังคุยกันเรื่องออกแบบ
6. เส้นสเก็ตช์หายไป เหลือแต่โลโก้
7. โลโก้เลื่อนไปมุมซ้ายบน แล้วเข้าหน้าเว็บหลัก

## ไฟล์ที่ต้องเตรียม
- SVG แยกชิ้น: "TWO", "DESK", Symbol TD → ลูกค้าเตรียมให้
- ภาพสเก็ตช์ → วาดเป็น SVG path ตามสไตล์ด้านล่าง

## สไตล์ลายเส้น (จาก Reference ที่ลูกค้าส่งมา)

### Symbol (ไอคอน เช่น บ้าน ตึก เครื่องมือ)
- ลายเส้นแบบ hand-drawn
- **ไม่แข็ง** — อยากได้เส้นเหมือน REF คน (ดู 01-06)
- วงกลมสีแดงในรูป = ไอคอนที่ถูกเลือก: ดินสอ+ไม้บรรทัด, หมวกก่อสร้าง, บ้าน, กล้องถ่ายรูป

### คน (figure)
- เส้นตัน ไม่บางมาก ไม่ยุกยิก
- เหมือนวาดด้วยปากกาเมจิก
- สไตล์ simple silhouette outline (ดู ref 01-06)
- ไม่ realistic — abstract แต่จำได้ว่าเป็นคน
- มีท่าทาง: นั่ง ยืน คุยกัน ชี้

**Why:** ลูกค้าต้องการ splash ที่สื่อตัวตนสตูดิโอ — เริ่มจากชื่อ รวมเป็น symbol แล้วเล่าว่าทำอะไร (ออกแบบ) ก่อนเข้าเว็บจริง
**How to apply:** ใช้ SVG path animation + Framer Motion, วาด illustration เป็น SVG inline เพื่อ animate stroke
